# 5️⃣ Anti-Cheat

***

### _<mark style="color:yellow;">Cette page décrit comment désactiver tous les logiciels antitriche sur votre PC.</mark>_

Ouvrez votre configuration système (appuyez sur Win+R -> tapez ms config -> appuyez sur Entrée) Accédez à l'onglet "Services"

#### Cochez la case "Masquer tous les services Microsoft"



* Cherchez dans la liste : Easy Anti-Cheat, BattlEye, etc.
* Décochez les services qui sont des anti-triches
* Cliquez sur appliquer et redémarrez votre PC

#### Valorant (Riot Vanguard), ESEA, FaceIT : Pour Riot Vanguard, ESEA, et FaceIT, vous devrez vous rendre dans vos Applications & Programmes dans les paramètres Windows pour les désinstaller complètement. Ils sont très intrusifs et empêcheront le pilote de se charger.
